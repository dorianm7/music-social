# Containers Introduction

## App.jsx
Where every part of the App is taken care of.  
Global CSS styles are placed in _App.css_.  
Routing will be done here.  

## InAppPage.jsx
Wraps components with the _AppNav_ and _Footer_ and adds a _h1_ heading.  
Specifically made to wrap _PageContents_ found within the App. 


## Pages Folder
Holds the different pages displayed in the App.  
Each route will display a new _Page_ from this directory.

## Page Contents Folder
Holds the contents that will be displayed in each page.
